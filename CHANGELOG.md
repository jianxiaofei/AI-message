# Change Log

All notable changes to the "AI-message" extension will be documented in this file.

## [0.0.22] - 2025-01-17

### Added
- 完整的 Git 和 SVN 双重支持
- GitHub Copilot 驱动的 AI 提交信息生成
- 智能表情符号支持（✨🐛📝💄♻️⚡等）
- 摘要行功能，显示所有变更文件
- ignore-on-commit 文件过滤功能
- 专业级 AI 提示工程
- 快捷键支持（Ctrl+Alt+G, Ctrl+Alt+Q）
- 右键菜单和 SCM 面板集成
- 智能后备方案（基于规则的提交信息）

### Fixed
- SVN 仓库检测和状态获取优化
- 文件差异分析改进
- 错误处理和用户体验优化

### Changed
- 升级到 VS Code 1.104.0+ 支持
- 简化配置选项，无需 API 密钥
- 依赖 GitHub Copilot 扩展

## [0.0.1] - 2025-01-01

### Added
- 初始版本发布
- 基础 SVN 提交信息生成功能
